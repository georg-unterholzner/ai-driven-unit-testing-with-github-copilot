# AI-Driven Unit Testing with GitHub Copilot - Example Repository

Example repository for the blog post [AI-Driven Unit Testing with GitHub Copilot](https://georg.dev/blog/06-ai-driven-unit-testing-with-github-copilot/).

This repository contains all the configuration files described in the article and a [todo-list app](./app/README.md) to try out the workflow.

Clone this repo to experiment with the AI-assisted unit testing workflow. See the article for detailed setup instructions and explanations.
